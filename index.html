Fill-Element Demo

Welcome to the Fill-Element Demo. This page will demonstrate and explain how the Fill-Element plugin works. In other words, this is both a demo and a README together.

About the Plugin

This plugin allows users divs to fill the parent element. You can choose the percentage of fill on the element as well as what margins you want on the object.

Usage

To use this plugin, you simply download it. When adding it to the site, add the script as "src = "path/fill.js"

At the bottom of your page, add the following code:

$(document).ready(function(){

$("element").fill({margin,percentWidth});

});

Resizing the Page

If you want this function to resize elements when the window size changes, simply call the function in $(window).resize() underneath the $(document).ready section

Sample Input

Demonstrating the Margins and Width

This grey div has been given a margin value of 5 and a width percentage(of the parent element) of 50. So the whole div will have a width of 50% of the parent element (in this case, the white box) and it will have a margin of 20px all around.

Tips on Usage

  • Using the window resize event handler allows the function to compensate for new window sizes.
  • Having all of your divs enclosed in a container with a width of 100% allows your divs that are resizing with the window to be flexible as well.
  • If you want to select a certain type of object such as all h1 elements on a page, put the .fill() function inside the arguments for the .each() function.
